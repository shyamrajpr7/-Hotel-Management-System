# 🏨 Hotel Management & Channel System

A high-availability hotel management system built using 
Java Spring Boot and MongoDB, designed to handle diverse 
room types, guest profiles, and real-time booking operations.

## 🚀 Tech Stack
- **Backend:** Java, Spring Boot 3.5
- **Database:** MongoDB
- **API:** REST API
- **Server:** Apache Tomcat (Embedded)

## ✨ Features
- Real-time booking management
- Multi-channel synchronization
- Flexible room type handling
- Guest profile management
- Document-oriented data model (no schema rigidity)

## ▶️ How to Run
1. Make sure MongoDB is running on `localhost:27017`
2. Clone the repo
3. Run the Spring Boot app
4. Open `http://localhost:8080`
